# Landing v3 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a new landing page for Arkadia Labs at `/:lang/v3` with 4 new creative components, 2 adapted sections, and 6 reused sections from the current site.

**Architecture:** Separate page (`LandingV3Page.tsx`) sharing Header/Footer via `PublicLayout`. New components in `src/components/landing/`. Translations in a `landingV3` i18next namespace. Reused components that need different copy accept an optional `translationNamespace` prop.

**Tech Stack:** React 18, TypeScript, Tailwind CSS, Framer Motion, i18next (HTTP backend, namespace-based), React Router v7, Vite.

**Design spec:** `docs/superpowers/specs/2026-04-08-landing-v3-design.md`
