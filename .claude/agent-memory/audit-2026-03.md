# SEO/AEO Audit - March 2026

## Scope Audited

Full site audit covering all pages, components, public assets, and translation files.

## Key File Paths

- `index.html` — static shell, no og/twitter meta
- `public/robots.txt` — no AI bots, wrong brand name
- `public/sitemap.xml` + `sitemap-en/es/cs.xml` — outdated, incomplete
- `src/pages/solutions/SolutionDetailPage.tsx` — only page with full Helmet + JSON-LD
- `src/pages/solutions/SolutionsIndexPage.tsx` — Helmet present, no JSON-LD, no og/twitter
- `src/components/FaqSection.tsx` — accordion FAQ, no FAQPage schema
- `src/data/solutions/lead-validator.ts` — has cited statistics (AEO positive)

## Confirmed Issues

1. Brand name mismatch: index.html = "Arkadia Labs", logo SVG = "arcadia_labs", LinkedIn = "elevaite-labs"
2. 12+ pages with zero Helmet meta tags
3. robots.txt missing all AI bot entries
4. Sitemaps: outdated lastmod, missing /solutions/* and /case-studies/* URLs, wrong URL pattern for case studies
5. SolutionDetailPage missing `cs` hreflang (has en + es + x-default only)
6. No Organization, WebSite, FAQPage schema anywhere
7. Footer "Case Studies" links to /#testimonials instead of /:lang/case-studies
8. Social media links for Twitter/YouTube/Instagram are placeholder `#` hrefs
9. Hero HeroFramed.tsx: subtitle uses `<h2>` tag (should be `<p>` since H1 is already on page)
10. Footer newsletter input has no label element
11. No skip-to-content accessibility link
12. No `x-default` hreflang in sitemaps
